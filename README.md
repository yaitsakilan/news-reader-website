# 📰 News Reader Website

A lightweight, responsive news portal simulation styled after classic digital newspapers like the *Times of India*. This project is built entirely with semantic HTML5 and vanilla CSS, demonstrating clean page layouts, navigation structures, and interactive dropdown menus without any external framework dependencies.

---

## 🚀 Getting Started

To launch and explore the website locally, follow these simple steps:

1. **Clone or Download the Repository:**
   ```bash
   git clone https://github.com/yaitsakilan/news-reader-website.git
   ```

2. **Open the Homepage:**
   Navigate into the `web` folder and open [mainpage.html](file:///d:/news-reader-website/web/mainpage.html) directly in any modern web browser (e.g., Google Chrome, Mozilla Firefox, Microsoft Edge, Safari).
   ```text
   web/mainpage.html
   ```

## 🛠️ Features & Highlights

- **Newspaper Typography & Layout:** Leverages distinct font styling, borders, letter-spacing, and alignments to recreate the look of print journalism.
- **CSS-only Dropdowns:** Pure CSS dropdown navigation for "About Us" and associated options, eliminating the need for JavaScript for UI controls.
- **Structured Data Tables:** Uses structured HTML tables with responsive image placement to balance the news grid.
- **Regional Portals:** Modularized HTML pages dedicated to city-specific news reportage.

---

## 🎨 Tech Stack

- **Structure:** Semantic HTML5
- **Style:** Vanilla CSS3
- **Media:** Local optimized images (`.jpeg`, `.jpg`) for headlines and branding
